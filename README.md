# Titanic Data Analysis - Internship Task 2

## Project Overview

This project analyzes the Titanic dataset using Python. The objective is to clean the data, analyze passenger survival patterns, and visualize the results using Matplotlib and Seaborn.

## Dataset

Titanic - Machine Learning from Disaster (Kaggle)

## Tools Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Steps Performed

1. Imported required libraries.
2. Loaded the Titanic dataset.
3. Explored the dataset.
4. Handled missing values.
5. Removed unnecessary columns.
6. Performed Exploratory Data Analysis (EDA).
7. Visualized survival by gender.
8. Visualized survival by passenger class.
9. Visualized age distribution.
10. Analyzed survival by age group.

## Key Findings

- Female passengers had a higher survival rate than males.
- First-class passengers had the highest survival rate.
- Children had a better survival rate than adults.
- Most passengers were between 20 and 40 years old.

## Conclusion

The project demonstrates data cleaning, data visualization, and exploratory data analysis using Python.
